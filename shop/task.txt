Есть магазин в нем есть отделы, а в отделах есть полки с товарами. 
Товары могут быть продовольственные и непродовольственные. 
Система должна позволить д обавить отдел, полку и товары, при условии что продовольственные товары не могут быть в непродовольственном отделе. 

Time: 2h
Code Language: Java
